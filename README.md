# Alfred-Find-emoji

## 预览
![](https://photos-picgo.oss-cn-beijing.aliyuncs.com/img/u0qxs-lqzql.gif)

## Features

+ 使用方法 `em {query}`
+ 一键调出搜索页面, 需要先在Alfred Workflow中设置热键
+ 按`Enter`键复制表情图标

## Dependencies

python2.7

## 安装

请移步Releases 下载最新版本即可
